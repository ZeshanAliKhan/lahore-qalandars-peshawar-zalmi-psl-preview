# Publish Guide

1. Create a public GitHub repository named `lahore-qalandars-peshawar-zalmi-psl-preview`.
2. Push this folder as the repository root.
3. Enable GitHub Pages from `Settings > Pages`.
4. Use source `Deploy from a branch`.
5. Select branch `main` and folder `/ root`.

Expected live URL:

`https://zeshanalikhan.github.io/lahore-qalandars-peshawar-zalmi-psl-preview/`
